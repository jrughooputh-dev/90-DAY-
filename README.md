# 90 Day — Personal Transformation System

A private health and fitness web app for a 90-day body recomposition programme.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Daily tracker — workouts, nutrition log, rest timer, journal |
| `report.html` | Full personal health report — body composition, risk assessment, diet and MPS protocol |
| `recovery.html` | Recovery, detox and self-improvement guide — smoking detox, lung recovery, muscle recovery, supplement stack, daily protocol |

## Features

- **Workout tracker** with exercise check-offs, phase progression (3 phases × 3 days), and YouTube video links for every exercise
- **Nutrition log** with daily macro tracking (protein, carbs, fat) — resets each day
- **Rest timer** with presets (30s to 30min) and phone vibration on completion
- **Session journal** with mood tagging
- **Phase progress bar** tracking Day 1–90 automatically from first open
- **Health report** with accurate body composition metrics, US Navy body fat calculation, full diet protocol, MPS and recovery science, lung nutrition, and psoriasis management
- **Recovery guide** with interactive checklists, smoking detox roadmap, supplement stack, and complete daily protocol

## Deployment

1. Upload all files to this repository
2. Go to **Settings → Pages → Source: main branch**
3. Your URL: `https://jrughooputh-dev.github.io/90day`

Add to iPhone home screen: Safari → Share → Add to Home Screen

## Data

All tracker data (workouts, nutrition, journal) saves to browser localStorage on your device. No server, no account, no data sent anywhere.
